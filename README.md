# 🧮 Calculadora Web

Este projeto consiste em uma **calculadora simples desenvolvida com HTML, CSS e JavaScript**, capaz de realizar as quatro operações matemáticas básicas.

O objetivo do projeto é praticar conceitos fundamentais de desenvolvimento web, como **estruturação de páginas, estilização e manipulação de lógica com JavaScript**.

---

## Funcionalidades

A calculadora permite realizar as seguintes operações:

- ➕ Soma  
- ➖ Subtração  
- ✖️ Multiplicação  
- ➗ Divisão  
- 🧹 Limpar os valores da calculadora

As operações são executadas utilizando **funções em JavaScript**, que manipulam os valores inseridos pelo usuário.

---

## Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

- **HTML5** → Estrutura da aplicação  
- **CSS3** → Estilização e layout da calculadora  
- **JavaScript** → Lógica das operações matemáticas

## ▶️ Como executar o projeto

Para utilizar o projeto em seu computador, siga os passos abaixo:

### 1️⃣ Clonar o repositório

```bash

```

### 2️⃣ Acessar a pasta do projeto

```bash
cd nome-do-repositorio
```

### 3️⃣ Executar o projeto

Basta **abrir o arquivo `index.html` no navegador**.

Não é necessário instalar nenhuma dependência.
